# Sejong University Capstone_Project

## 주인을 따라다니며 셀카찍는 드론
> 기존 셀카방식의 불편함을 타파하고자 주인을 추적하며 셀카찍는 드론을 개발
### 특징
1. 얼굴인식기술을 활용한 주인 인식
2. 제스처인식 또는 UI를 통한 간편한 드론 제어
3. 실시간 사용자 스위칭
4. 사용자 제스처-기능 매핑
5. 웹을 이용한 드론 제어
